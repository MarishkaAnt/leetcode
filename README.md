# Решение задач на алгоритмы (LeetCode, CodeRun, Yandex)

В данном репозитории представлены мои решения алгоритмических задач.

После того как мое решение было принято платформой, я смотрю более эффективные и быстрые решения других участников 
или обращаюсь к Copilot для оптимизации моего решения, поэтому у меня, как правило, по несколько сабмитов в задачах. 
В коммит сюда отправляется последняя версия решения, которая устроила меня по сочетанию быстродействия и затрат памяти. 
Если мне требуется дополнительное пояснение используемого алгоритма или есть решение, 
которое я хотела бы оставить для примера на будущее, то данные заметки помещаются в комментарии под кодом.

В каждом классе перед кодом я указываю номер задачи и ее название 
(точно такое же как в LeetCode, CodeRun, Yandex для удобного поиска), 
описываю условие задачи и примеры верного ввода-вывода, а также веду разработку методом TDD, 
поэтому для каждого класса написаны тесты.
